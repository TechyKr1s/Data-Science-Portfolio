# 🍿 Netflix Movies Analysis

This project explores the Netflix Movies and TV Shows dataset to uncover trends in content production, ratings, genres, duration and country contributions.

The analysis focuses on understanding:
- the distribution of movies vs TV shows
- Netflix content growth over time
- dominant producing countries
- common genres and ratings
- movie and TV show duration statistics

The project was developed using Python, Pandas and Matplotlib inside Jupyter Notebook.

---

## 📊 Dataset

The dataset contains information about Netflix titles, including:
- title
- type
- director
- cast
- country
- release year
- rating
- duration

---

## 🎯 Objectives

The main objectives of this analysis were:

- Compare the number of Movies vs TV Shows
- Analyze Netflix content growth over time
- Identify countries producing the most Netflix content
- Explore ratings distribution and audience focus
- Analyze duration statistics
- Discover the most common genres and directors

---

## 🧹 Data Cleaning & Preprocessing

Several preprocessing steps were performed:

- Removed missing values
- Renamed columns for readability
- Split multiple countries into separate rows
- Split genres into separate rows
- Split directors into separate rows
- Converted date columns to datetime format
- Extracted numerical duration values
